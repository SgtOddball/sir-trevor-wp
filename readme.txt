Sir Trevor JS Post Editor for Wordpress. An intuitive, block-based content editor for Wordpress.

== Installation ==

1. Upload `sir-trevor-wp` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I still use the normal editor? =

Yes. To create a new post with the normal editor, click the "use code editor" button. You cannot edit posts created with
this editor in the normal editor. Similarly, you cannot edit posts created in the normal editor in this editor.
